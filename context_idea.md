## 핵심 차별점

1. **영수증 인증 리뷰** — 네이버/카카오에 없음. 인증 뱃지 + 상위 노출
2. **반려동물이 주인공인 UI** — "보리가 다녀온 병원" 프레임. 사람이 아닌 아이 이름/사진이 먼저
3. **진료비 투명성** — 실제 영수증 기반 진료비 데이터. 맘카페에서 항상 물어보는 것
4. **다항목 별점** — 진료비 수준 / 의사 친절도 / 시설 청결도 / 대기시간 / 종합 만족도 (성예사 방식)

> ⚠️ 영수증 인증이 차별점이라면, 그 인증이 실제로 작동해야 함.
> 검증 로직이 허술하면 차별점 자체가 신뢰도 폭탄이 됨. 콘텐츠 수집보다 인증 신뢰성이 먼저.

## 포지셔닝 문제 (근본 이슈)

- 네이버 플레이스가 동물병원 + 미용샵 리뷰를 이미 하고 있음
- "왜 굳이 여기서 리뷰를 봐야 하나?" — 방문자에게 10초 안에 차별점이 체감돼야 함
- 영수증 인증이 차별점이지만, 리뷰가 없으면 인증 시스템도 빛을 못 발하는 악순환
- 콘텐츠 없음 + 차별점 불명확 + 개인 프로젝트 느낌 → 반박할 수 없는 현실

## 개선 우선순위

### 1순위 — 지금 당장

#### 리디자인
- 탭 기반 네비게이션 (홈 / 후기 / 커뮤니티 / 마이)
- 흰 배경 + 그린 포인트 1개로 컬러 통일
- 홈 진입 시 최근 후기 피드 바로 노출
- 반려동물 이름/사진이 리뷰 카드 주인공

#### 총 진료비 범위 표시
- 지금은 1회 방문 기준 범위로 표시
- 예: `₩10,000 ~ ₩70,000 / 1회 방문`
- 항목별 평균 데이터는 나중에 (데이터 축적 필요)

#### 다항목 별점
- 진료비 수준 / 의사 친절도 / 시설 청결도 / 대기시간 / 종합 만족도
- 레이더 차트 또는 항목별 바 시각화
- "이 병원은 친절한데 비싸다" 같은 인사이트 제공

#### 커뮤니티 게시판
- 질문/추천 / 반려동물 자랑 / 정보 공유
- 글 태그: 병원추천 / 질문 / 자랑 / 정보
- 인증 없이도 글 작성 가능 (진입 장벽 낮추기)

#### 인증 없는 일반 후기 허용
- 영수증 인증 = 인증 뱃지 표시
- 일반 후기도 허용 → 초기 콘텐츠 확보 속도 향상

### 2순위 — 다음 작업

- 진료항목별 평균 진료비 표시 (핵심 차별점)
- 반려동물 프로필 고도화 (종, 성별, 나이, 등록번호)
- 리뷰 사진 첨부
- 반려동물 종별 필터 (강아지 / 고양이 / 기타)
- 병원 야간진료 여부 표시
- 마이펫 인증 뱃지

### 3순위 — 커뮤니티 활성화 이후

- 우리 동네 병원 랭킹 (위치 기반 + 다항목 별점)
- 이달의 베스트 리뷰어 (관리자 선정 → 메인 노출)
- 병원 사장님 답변 기능
- 수의사 인증 뱃지
- 유기견 입양 / 임보 게시판 (커뮤니티 서브 카테고리)

### 4순위 — 장기 로드맵

- 병원 없으면 직접 입력 옵션
- 병원 상단 노출 유료화 (트래픽 생기면)
- PWA → 네이티브 앱
- 카카오 / 네이버 로그인
- 반려동물 용품 쇼핑 탭
